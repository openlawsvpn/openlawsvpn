Install

amd64
curl -LO https://github.com/openlawsvpn/openlawsvpn/releases/latest/download/openlawsvpn-cli-linux-amd64
chmod +x openlawsvpn-cli-linux-amd64
sudo mv openlawsvpn-cli-linux-amd64 /usr/local/bin/openlawsvpn-cli
arm64
curl -LO https://github.com/openlawsvpn/openlawsvpn/releases/latest/download/openlawsvpn-cli-linux-arm64
chmod +x openlawsvpn-cli-linux-arm64
sudo mv openlawsvpn-cli-linux-arm64 /usr/local/bin/openlawsvpn-cli

static binary runs in standalone (root) mode only — requires root or CAP_NET_ADMIN+CAP_NET_RAW; rootless mode is available via the Fedora / RHEL packages

as root
dnf copr enable vorona/openlawsvpn -y
dnf install openlawsvpn-cli -y
openlawsvpn-cli connect /path/to/client-config.ovpn --standalone
rootless
sudo dnf copr enable vorona/openlawsvpn -y
sudo dnf install openlawsvpn-cli openvpn3 -y
openlawsvpn-cli connect ~/Downloads/client-config.ovpn

rootless mode requires openvpn3 and its dependency gdbuspp

git clone https://github.com/openlawsvpn/openlawsvpn
make linux
